Система интеллектуального распределения задач
Веб-приложение для автоматического распределения задач в IT-команде на основе анализа компетенций сотрудников и их текущей загрузки. Используется венгерский алгоритм.

Локальный запуск
streamlit run app.py
После запуска приложение будет доступно по адресу: http://localhost:8501

Данные
Данные о сотрудниках и задачах хранятся в JSON-файлах в папке data/:

employees.json — сотрудники (навыки, доступное время)

tasks.json — задачи (требуемые навыки, трудозатраты)

Вы можете редактировать эти файлы вручную или через интерфейс приложения.

Установка зависимостей
bash
pip install -r requirements.txt
Основные библиотеки: streamlit, pandas, numpy, scipy, matplotlib, plotly, seaborn

Краткое описание работы
Тимлид загружает данные о сотрудниках и задачах.

Система строит матрицу компетенций (доля совпавших навыков).

Венгерский алгоритм находит оптимальное распределение.

Результат отображается в виде таблицы, тепловой карты и диаграмм.

При необходимости тимлид может вручную скорректировать назначения.

Поддерживаются два критерия оптимизации:

Максимум компетенций — приоритет качества назначения.

Сбалансированная загрузка — учёт текущей занятости сотрудников.
