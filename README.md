# Система интеллектуального распределения задач
## Веб-приложение для автоматического распределения задач в IT-команде на основе анализа компетенций сотрудников и их текущей загрузки. Используется венгерский алгоритм.

### Локальный запуск
```bash
streamlit run app.py
```
После запуска приложение будет доступно по адресу: http://localhost:8501

### Данные
Данные о сотрудниках и задачах хранятся в JSON-файлах в папке data/:

• employees.json — сотрудники (навыки, доступное время)<br>
• tasks.json — задачи (требуемые навыки, трудозатраты)

Вы можете редактировать эти файлы вручную или через интерфейс приложения.

### Установка зависимостей
```
pip install -r requirements.txt
```
Основные библиотеки: streamlit, pandas, numpy, scipy, matplotlib, plotly, seaborn

### Структура проекта
```
task_distributor/
├── app.py                 # Главный файл (Streamlit-приложение)
├── core.py                # Ядро системы (класс TaskDistributorCore)
├── data/
│   ├── employees.json     # Данные о сотрудниках
│   └── tasks.json         # Данные о задачах
├── history.json           # История распределений (создаётся автоматически)
├── assigned_tasks.json    # Назначенные задачи (создаётся автоматически)
├── requirements.txt       # Зависимости
└── README.md
```
### Краткое описание работы

• Тимлид загружает данные о сотрудниках и задачах.<br>
• Система строит матрицу компетенций (доля совпавших навыков).<br>
• Венгерский алгоритм находит оптимальное распределение.<br>
• Результат отображается в виде таблицы, тепловой карты и диаграмм.<br>
• При необходимости тимлид может вручную скорректировать назначения.<br>

### Поддерживаются два критерия оптимизации:

• Максимум компетенций — приоритет качества назначения.<br>
• Сбалансированная загрузка — учёт текущей занятости сотрудников.
